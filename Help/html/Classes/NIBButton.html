NIBButton Class Reference

Inherits from UIButton

Overview

NIBButton is a class that initializes a button of the calculator and assigns a tag of NIBButtonTag for the button as well as set some custom effects of the button. Based on the type of the button, the class may draw extra parts for the buttons when necessary.
