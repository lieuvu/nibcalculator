NIBOperator Class Reference

Inherits from NSObject
Declared in NIBOperator.h

Overview

NIBOperator is a class to wrap the operators of calculation and provide some utilities methods to query the type of operators or compare the priority of two operators.

Note: The class can not be instantiated and will generate an error if doing so. The class includes only class methods.
